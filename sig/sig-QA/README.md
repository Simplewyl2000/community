# QA SIG
openEuler QA(质量保障)项目团队致力于如何更好的提升openEuler社区的质量保证，不仅局限在社区版本的测试，
质量保障项目主要涉及的质量保障活动包括：
- 开源当前具有的测试能力，让更多的社区开发者进行使用、优化并贡献
- 测试验证openEuler发布版本并评估其质量
- 问题分类，将问题分配到对应的项目并跟踪解决
- 开发测试工具以提升代码开发效率和测试验证效率
- 制定、发布和维护管理版本发布标准及流程
