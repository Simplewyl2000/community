# OKD SIG

## 使命和愿景

OKD是红帽公司开源的企业级容器云PaaS平台，用以管理混合云和多云应用部署。

您可以访问 OKD 的[官网](https://www.okd.io/) 获得关于 OKD 的更多信息。

## SIG组职责

- 在openEuler社区中添加对 OKD 的支持
- 负责 OKD 相关软件包的规划、维护和升级
- 及时响应用户反馈，解决相关问题


# 组织会议
