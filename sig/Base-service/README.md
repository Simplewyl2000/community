# Base-Service
- 维护常见的开发构建工具，例如gcc、make、m4、git 等 
- 维护系统的基础运行环境库，例如d-bus,systemd,cronie 等
- 维护部分常见应用，例如vim,zip 等 

# 组织会议
- 公开的会议时间：NA
- 邮件讨论为主，涉及决策的内容，TC会议申报议题
