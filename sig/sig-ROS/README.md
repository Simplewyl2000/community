# openEuler ROS SIG

- 在openEuler社区中添加对ROS的支持
- 根据openEuler迭代版本，持续完成ROS中各个组件向openEuler的移植，并提供ROS在openEuler上的使用文档
- 及时响应用户反馈，解决相关问题
