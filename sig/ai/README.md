# AI SIG

- 打造数字基础设施AI生态链，协同芯片厂商、第三方AI框架等一起发展AI产业，包括AI+大数据/HPC/搜索等融合。 
- 提供内置的AI框架训练/推理软件，让生态伙伴能够快速基于内置的AI能力开展AI相关技术探索。
- 构建AI相关软件维护能力，保障上游开源软件的供应安全以及未来的演进。
- 协同MindSpore等社区+openEuler社区，共同致力于开发者生态的繁荣。
