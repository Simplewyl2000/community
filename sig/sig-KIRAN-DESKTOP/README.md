# KIRAN-DESKTOP SIG

    KIRAN-DESKTOP，基于Mate-Desktop开发，专门为图形化服务器操作设计的桌面环境，操作简洁，为系统管理员提供对服务器高效的操作方式。

#职责
 
- 在openEuler社区中添加对 KIRAN-DESKTOP  的支持
- 负责 KIRAN-DESKTOP  相关软件包的规划、维护和升级
- 及时响应用户反馈，解决相关问题

# 组织会议

- 公开的会议时间：北京时间，每双周三 下午，16：00~17：00。会提前提醒通知会议，如遇时间冲突，可研究延迟会议。
