
# sig-buildsystem

* sig-buildsystem 旨在为 OpenEuler 打造一个用于编译 RPM 的编译系统，努力构造一个比较完善的，基于原生RPM分布式编译系统的 CI-loop 和 验证测试环境。
* Koji 原是 Fedora 社区的编译系统，它使用 Mock 创建编译软件包时需要的编译环境，所以本 SIG 将以完成 Koji for OpenEuler为起点
