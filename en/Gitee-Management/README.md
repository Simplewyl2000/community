# Gitee 管理

openEuler项目使用Gitee来存储和组织代码，管理问题和文档，并提供一致的贡献者流程。为了简化Gitee存储库的组织和管理。我们提供了一些工具来自动设置和执行管理策略。

这些策略和工具的支持由基础设施下的Gitee管理团队负责。
